Wipe plugin for Total Commander
----------------------------------------
version 0.2

File destroyer (secure erase) plugin.
Wipes files according to Department of Defense standard DOD 5220.22-M.


How to install the plugin in Total Commander:
 
1. Unzip the archive to a directory
2. Choose Configuration - Options - Packer - Config. Packer Extension DLLs
3. Type wipe as new extension
4. Click new type, and select wipe.wcx
5. Click OK.
